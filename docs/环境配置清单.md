# 环境配置清单

> 在运行系统之前，请按此清单逐项完成环境配置。

---

## 1. 硬件要求

| 项目 | 最低要求 |
|------|----------|
| CPU | 4 核 |
| 内存 | 8 GB |
| 硬盘 | 100 GB（系统占用约 10GB） |
| 网络 | 首次启动需连接互联网下载模型 |

---

## 2. 软件安装

### 2.1 Python 3.10+

```bash
# 检查版本
python --version   # Windows
python3 --version  # Linux/macOS

# 下载地址: https://www.python.org/downloads/
# 安装时勾选 "Add Python to PATH"
```

### 2.2 Node.js 18+

```bash
# 检查版本
node --version
npm --version

# 下载地址: https://nodejs.org/
# 推荐 LTS 版本
```

### 2.3 Ollama

```bash
# 检查版本
ollama --version

# Windows: https://ollama.com/download
# Linux: curl -fsSL https://ollama.com/install.sh | sh
# macOS: https://ollama.com/download
```

---

## 3. 安装步骤

### 3.1 拉取模型文件（首次，需联网，约 5GB）

```bash
# 大语言模型（约 4.5GB）
ollama pull qwen2.5:7b

# 嵌入模型（约 274MB）
ollama pull nomic-embed-text

# 验证模型
ollama list
```

### 3.2 后端依赖

```bash
cd backend
pip install -r requirements.txt
```

依赖清单：
```
fastapi, uvicorn, sqlalchemy, pydantic, pyjwt, bcrypt,
python-multipart, chromadb, langchain, langchain-community,
requests, jieba, openpyxl
```

### 3.3 前端依赖（可选，生产模式不需要）

```bash
cd frontend
npm install
```

### 3.4 生产模式构建（可选）

```bash
cd frontend
npm run build
# 构建产物输出到 frontend/dist/
# 后端会自动托管静态文件
```

---

## 4. 启动系统

### 方式一：使用启动脚本（推荐）

```bash
# Windows
start.bat

# Linux/macOS
chmod +x start.sh
./start.sh
```

脚本会自动：
1. 检查并启动 Ollama 服务
2. 检查并下载模型（如未安装）
3. 安装 Python 依赖
4. 启动后端服务（含种子数据初始化）
5. 输出访问地址和默认账号

### 方式二：手动启动

```bash
# 终端1：启动 Ollama
ollama serve

# 终端2：启动后端
cd backend
python main.py

# 终端3：启动前端（可选）
cd frontend
npm run dev
```

## 8. 常见问题

### Q: Ollama 连接失败
```bash
# 确认 Ollama 在运行
ollama list

# 如未运行
ollama serve
```

### Q: ChromaDB 报错
```bash
# 删除向量库重新初始化
rm -rf backend/data/chromadb/
# 重启后端会自动重建
```

### Q: 端口被占用
```bash
# Windows: 查看端口占用
netstat -ano | findstr :8000

# Linux: 
lsof -i :8000
```

### Q: 内存不足
```bash
# 建议关闭其他大内存应用
# 生产模式可省去前端开发服务器（约 0.3GB）
```
